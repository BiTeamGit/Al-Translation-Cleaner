# AL Translation Cleaner
[![License: MIT](https://img.shields.io/github/license/BiTeamGit/Al-Translation-Cleaner)](https://opensource.org/licenses/MIT) ![Visual Studio Marketplace Version](https://img.shields.io/github/v/release/BiTeamGit/Al-Translation-Cleaner)


This is the repository for AL Translation Cleaner, a Visual Studio Code extension. Learn more about AL Translation Cleaners functions and capabilities in the extension [README](extension/README.md).

Check it out on [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=biteam.al-translation-cleaner). If you like it please consider leaving a [review](https://marketplace.visualstudio.com/items?itemName=biteam.al-translation-cleaner&ssr=false#review-details)

## Installation
Launch VS Code Quick Open (Ctrl+P), paste the following command, and press enter.
```
ext install biteam.al-translation-cleaner
```
